# middle-ages-blog
Small Vue app with tests for QA workshop. Uses Jest. Has integration tests.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run tests
```
npm run test
npm run test:watch
```
